# Flight Satisfaction Prediction

A Machine Learning project to predict whether an airline passenger is satisfied or not based on flight service data.

## Project Overview

This project analyzes airline passenger feedback and builds a classification model to predict customer satisfaction. It includes data preprocessing, exploratory data analysis, model training, and prediction using a Flask web application.

## Features

- Data cleaning and preprocessing
- Exploratory Data Analysis
- Machine Learning classification model
- Flask-based web application
- Passenger satisfaction prediction
- User-friendly interface

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- LightGBM
- Flask
- HTML/CSS
