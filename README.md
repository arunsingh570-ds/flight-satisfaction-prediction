# Flight Satisfaction Prediction
This project predicts whether an airline passenger is satisfied or not using machine learning based on flight service data.
## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Model training and evaluation
- Passenger satisfaction prediction
## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
## Objective
To build a machine learning model that can predict customer satisfaction from airline-related data.
